### Prerequisites

You first need to install manim
* pip
  ```sh
  pip install manim
  ```

Additionally, you may need to install tex

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/Sentientplatypus/circle-area-proof.git
   ```
